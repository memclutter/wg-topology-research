# Point to Point

Target nodes:

- VPS with ssh access, >= 1CPU, 512Mb (tested on Ubuntu 22.04, 64bit)
- VPS with ssh access, >= 1CPU, 512Mb (tested on Ubuntu 22.04, 64bit)

Local machine:

- Linux machine with ansible installed
